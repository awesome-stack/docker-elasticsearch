# Docker compose template for Elasticsearch and Redash

## Requirements
* Docker
* Docker Compose

## Build Docker Image
```
sh build.sh
```

## Run
```
docker-compose up -d
```
* Redash: http://localhost:8081/
* Elasticsearch: http://localhost:9200
